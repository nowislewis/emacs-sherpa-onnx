#+title: emacs-sherpa-onnx
#+language: en

Streaming dictation into Emacs with [[https://github.com/k2-fsa/sherpa-onnx][sherpa-onnx]] running a streaming
Zipformer2-CTC (ONNX, int8) — Chinese + English ASR with real-time
output.  Pure CPU, offline, no server, no GPU.

*How it works*

The streaming model produces incremental text every ~200 ms while
you speak; it appears in Emacs as a grey overlay and is committed to
the buffer at each pause.  The inserted text is *raw* (no punctuation,
no error correction) — run your own LLM command on the region
afterwards to add punctuation, fix homophones, and polish.

*Requirements*

- [[https://docs.astral.sh/uv/][uv]], =ffmpeg=, ~160 MB disk (model)

*Quick start*

#+begin_src shell
make install      # venv + download streaming Zipformer2-CTC (~120 MB)
make test-stream  # transcribe a bundled test wav
#+end_src

#+begin_src emacs-lisp
(add-to-list 'load-path "~/.emacs.d/lib/emacs-sherpa-onnx")
(require 'emacs-sherpa)
(global-set-key (kbd "C-c d") #'emacs-sherpa-dictate)
#+end_src

| Command                    | Description                                  |
|----------------------------+----------------------------------------------|
| =emacs-sherpa-dictate=       | Toggle streaming dictation (mic → text)     |
| =emacs-sherpa-dictate-file=  | Transcribe an existing WAV file             |
| =emacs-sherpa-cancel=        | Cancel, discard partial text                |
| =emacs-sherpa-only-record=   | Record mic → save WAV (no ASR)             |
| =emacs-sherpa-install=       | Download models                             |
| =emacs-sherpa-map=           | Prefix keymap: d / c / r                   |

*Customization*

| Variable                           | Default     |
|------------------------------------+-------------|
| =emacs-sherpa-ffmpeg-input-format=   | =pulse=       |
| =emacs-sherpa-ffmpeg-input-device=   | =default=     |
| =emacs-sherpa-recordings-directory=  | =~/Downloads= |
| =emacs-sherpa-python=              | auto        |

*CLI*

#+begin_src shell
# Mic streaming (via ffmpeg pipe)
ffmpeg -f pulse -i default -ar 16000 -ac 1 -f s16le pipe:1 | asr-sherpa-stream

# File transcription
asr-sherpa-stream --wav audio.wav

# Environment
ASR_NO_ENDPOINT=1   # disable endpoint detection (pipe mode)
#+end_src

*Post-processing (optional)*

Inserted text is raw.  To clean it up, run any LLM command on the
region — e.g. with gptel, ellama, or a shell filter.  Example prompt:
"给下面的语音转写加上标点，修正明显的同音字错误，不要改变原意".
